. ส่วนฮาร์ดแวร์ (Hardware)
ประกอบด้วยอุปกรณ์ดังนี้
	ESP32 : บอร์ดหลักใช้ควบคุมการทำงานทั้งหมดของระบบ และเชื่อมต่อกับอินเทอร์เน็ตผ่าน Wi-Fi
	DHT11 : เซนเซอร์ตรวจวัดอุณหภูมิและความชื้นในอากาศ
	LDR : เซนเซอร์ตรวจวัดระดับแสงเพื่อควบคุมไฟอัตโนมัติ
	PIR Sensor : เซนเซอร์ตรวจจับการเคลื่อนไหวในพื้นที่
	LED และ RGB LED : แสดงสถานะการทำงานของระบบ เช่น ไฟเตือนหรือแสงสว่าง
	Buzzer : แจ้งเตือนเมื่อมีความผิดปกติ เช่น ความร้อนสูงหรือมีการเคลื่อนไหวผิดปกติ

2. ส่วนซอฟต์แวร์ (Software)
	Arduino IDE : ใช้เขียนโปรแกรมและอัปโหลดโค้ดไปยังบอร์ด ESP32
	Firebase Realtime Database : ใช้จัดเก็บข้อมูลจากเซนเซอร์และสถานะอุปกรณ์
	Mobile Application : ใช้ควบคุมและแสดงผลข้อมูลแบบเรียลไทม์บนโทรศัพท์มือถือ


	ขั้นตอนการติดตั้งและใช้งาน
ขั้นตอนที่ 1: การติดตั้งฮาร์ดแวร์
	เชื่อมต่อเซนเซอร์แต่ละตัวเข้ากับขาของ ESP32 ดังนี้
	DHT11 → ขา 23
	LDR → ขา 33
	PIR Sensor → ขา 15
	LED แดง → ขา 18
	LED เขียว → ขา 19
	RGB (แดง, เขียว, น้ำเงิน) → ขา 25, 26, 27
	Buzzer → ขา 5
	ต่อสาย GND และ VCC ให้ครบทุกอุปกรณ์ (ใช้แรงดัน 3.3V หรือ 5V ตามชนิดของเซนเซอร์)
	ตรวจสอบการต่อสายให้ถูกต้องก่อนอัปโหลดโปรแกรม

ขั้นตอนที่ 2: การตั้งค่าโปรแกรมใน Arduino IDE
	เปิดโปรแกรม Arduino IDE
	ติดตั้ง ESP32 Board Manager โดยเพิ่ม URL:
	https://dl.espressif.com/dl/package_esp32_index.json
	เลือกบอร์ดเป็น “ESP32 Dev Module”
	ติดตั้งไลบรารีต่อไปนี้:
	WiFi.h
	Firebase_ESP_Client.h // Firebase ESP Client
              DHT sensor library (ของ Adafruit)
	Adafruit Unified Sensor
	DHT.h
	วางโค้ดที่ใช้ (จากโค้ดโครงงานของคุณ) ลงใน Arduino IDE
	แก้ไขค่า Wi-Fi และ Firebase ให้ตรงกับของผู้ใช้:
	#define WIFI_SSID "ชื่อ WiFi ของคุณ"
	#define WIFI_PASSWORD "รหัสผ่าน"
	#define DATABASE_URL "URL ของ Firebase"
	#define DATABASE_SECRET "Secret Key ของ Firebase"
	อัปโหลดโค้ดไปยังบอร์ด ESP32

ขั้นตอนที่ 3: การตั้งค่า Firebase
	เข้าเว็บไซต์ https://console.firebase.google.com
	สร้างโปรเจกต์ใหม่ → ไปที่ Realtime Database
	เปิดโหมด “Test Mode” เพื่อให้ ESP32 เขียนข้อมูลได้
	สร้างโครงสร้างข้อมูลตัวอย่าง เช่น
	/sensors/
	   temperature
	   humidity
	   light
	   motion
	/settings/
	   temperature_limit
	   humidity_limit
	   light_limit
	/rgb/
	   red
	   green
	   blue
	/systemOn
	คัดลอก Database URL และ Secret Key ไปใส่ในโค้ด ESP32

ขั้นตอนที่ 4: การใช้งานแอปพลิเคชัน
	เปิดแอปสมาร์ทโฮมบนโทรศัพท์มือถือ (พัฒนาโดยเชื่อม Firebase เดียวกัน)
	หน้าแรกจะแสดงข้อมูลจากเซนเซอร์แบบเรียลไทม์ เช่น อุณหภูมิ ความชื้น แสง และการเคลื่อนไหว
	ผู้ใช้สามารถ:
	สั่งเปิด–ปิดอุปกรณ์ไฟฟ้า
	เปลี่ยนค่าจำกัด (Threshold) เช่น ความร้อนสูงสุด
	เปิด–ปิดไฟ RGB ได้ทันทีจากหน้าจอ
	เมื่อมีเหตุการณ์ผิดปกติ เช่น อุณหภูมิสูงเกิน ระบบจะแจ้งเตือนขึ้นในแอปทันที
